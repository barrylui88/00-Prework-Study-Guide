HTML

  • The head element contains information about the webpage.
  • Head elements are not typically visible to the user as it contains metadata useful to the browser.
  • The body element represents the visible content shown to the user.
  • There are a number of different child elements that can sit under the body element which will give the webpage some structure.

CSS

  • To add style to our webpage, we will use Cascading Style Sheets (CSS)
  • This is a programming language used by developers to define how webpage content is presented to users.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.

Git

  • Git is a free, open source version control system that allows web developers to collaborate on projects without overwriting each other's work.
  • It also allows developers to revert to previous versions of the code if a problem is discovered.
  • GitHub is the largest provider of internet hosting for software and web development code, and provides the version control functionality of Git in a central repository.
